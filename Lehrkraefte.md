# Hinweise für Lehrkräfte

Dieses GitHub-Repository stellt begleitende Materialien zum Buch
**„SQL & Python in der Praxis: Datenbank-Management mit SQLite“** bereit.

Die Nutzung der Materialien ist **ohne Vorkenntnisse zu GitHub oder Git**
möglich.

---

## Materialien herunterladen (empfohlen)

1. Öffnen Sie das Repository:
   https://github.com/kwschneider/datenbanken-grundlagen

2. Klicken Sie auf **„Code“** (grüner Button)

3. Wählen Sie **„Download ZIP“**

4. Entpacken Sie die ZIP-Datei auf Ihrem Rechner

Sie können die enthaltenen Dateien anschließend wie gewohnt im Unterricht
verwenden oder an Lernende weitergeben.

---

## Struktur der Materialien

Die Inhalte sind kapitelweise aufgebaut und entsprechen dem Buch:

- Jeder Ordner steht für ein Kapitel
- SQL-Dateien enthalten kommentierte Beispiele
- Ein Ordner **`loesungen`** enthält Musterlösungen zu ausgewählten Aufgaben

---

## Nutzung im Unterricht

- Die Materialien dürfen frei verwendet, angepasst und weitergegeben werden
- Eine kommerzielle Nutzung (z. B. in der Weiterbildung) ist erlaubt
- Voraussetzung ist lediglich die Nennung des Urhebers

Die Codebeispiele sind bewusst einfach gehalten und für den Einsatz
in Ausbildung und Lehre konzipiert.

---

## Technische Hinweise

- Die SQL-Dateien (.sql) sind mit gängigen Datenbanksystemen nutzbar
  (z. B. SQLite, MySQL, MariaDB)
- Die Python-Dateien (.py) sind mit allen gängigen Software-Entwicklungsumgebungen
  nutzbar (z.B. Visual Studio Code, IDLE, Eclpise)

---

## Rückmeldungen und Fehlerhinweise

Hinweise, Korrekturen oder Verbesserungsvorschläge können über die
**Issues-Funktion** von GitHub gemeldet werden.

Alternativ können Sie das Repository auch lediglich als
Downloadquelle verwenden.
