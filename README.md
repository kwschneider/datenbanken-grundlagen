# datenbanken-grundlagen
Dieses Repository enthält alle Codebeispiele, Übungen und Zusatzmaterialien
zum Buch **„SQL &amp; Python in der Praxis: Datenbank-Management mit SQLite“** von Klaus-Werner Schneider.

## Zielgruppe
- Berufsschüler:innen der IT-Berufe
- Studierende im Grundlagenbereich
- Selbstlernende

## Struktur
Die Materialien sind kapitelweise organisiert und entsprechen dem Aufbau des Buches.

## Nutzung
Die SQL-Statements können mit gängigen relationalen Datenbanksystemen
(z. B. SQLite, MySQL, MariaDB) verwendet werden.

Die Lizenz wird im Repository angegeben.
